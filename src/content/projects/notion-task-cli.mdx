This project mainly was my way of learning golang with real projects instead of just doing the basic "hello world".

The idea of this project is to create a command-line tool that can quickly add tasks to a notion database. I wanted to integrate it with flow launcher so that I can quickly add tasks without having to open notion nor opening a terminal.

**Disclaimer**: the project has not been maintained for a while, since I've changed my os to linux, and flow launcher is only available on windows--also, I stopped using notion, but the cli tool itself should still work fine.
