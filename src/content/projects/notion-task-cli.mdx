this project mainly was my way of learning go with real projects instead of just doing the basic "hello world".

the idea of this project is to create a command-line tool that can quickly add tasks to a notion database. i wanted to integrate it with flow launcher so that i can quickly add tasks without having to open notion nor opening a terminal.

**disclamer**: the project has not been mantained for a while, since i've change my os to linux, and flow launcher is only available on windows--also, i stopped using notion, but the cli tool itself should still work fine.
